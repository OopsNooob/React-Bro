# React-Bro

A React learning and practice project built with **React 19** and **Vite**.

## Overview

This project is a collection of React component exercises covering core concepts such as state management, hooks, props, conditional rendering, and more. Each component demonstrates a specific React pattern or feature.

## Tech Stack

- [React 19](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [prop-types](https://github.com/facebook/prop-types)
- ESLint (with react-hooks and react-refresh plugins)

## Components

| Component | Description |
|-----------|-------------|
| `Stopwatch` | Functional stopwatch with start, stop, and reset controls using `useState`, `useEffect`, and `useRef` |
| `ToDoList` | To-do list with add, delete, and reorder (move up/down) functionality |
| `Counter` | Click counter with increment, decrement, and reset buttons |
| `DigitalClock` | Live digital clock |
| `ColorPicker` | Interactive color picker |
| `UserGreeting` | Conditional greeting based on user login state |
| `List` | Rendered list of items |
| `Card` | Reusable card UI component |
| `Button` / `Button2` | Reusable button components |
| `Header` / `Footer` | Page layout components |
| `ProfilePicture` | Profile image component |
| `Student` | Student info display component |
| `Food` | Food item display component |
| `ComponentA–D` | Component composition and props drilling examples |
| `MyComponent–MyComponent8` | Miscellaneous practice components |

## Getting Started

```bash
# Install dependencies
cd my-react-app
npm install

# Start the development server
npm run dev

# Build for production
npm run build

# Preview the production build
npm run preview

# Lint the code
npm run lint
```

The app runs on `http://localhost:5173` by default.
